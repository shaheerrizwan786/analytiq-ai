## Setup

1. Clone the repository
2. Open in VS Code

### Environment Variables

Create a `.env.local` file at the root and copy the contents of `.env.example`:

```
CLAUDE_API_KEY=
GOOGLE_API_KEY=
YELP_API_KEY=
APIFY_API_KEY=
DATABASE_URL=
```
